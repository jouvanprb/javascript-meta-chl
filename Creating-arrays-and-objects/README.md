# Latihan: Membuat array dan objek

Dalam lab latihan ini Anda akan berlatih membuat array dan objek.

## Tugas 
1. Buat sebuah larik kosong literal baru dan tetapkan ke variabel clothes.
2. Tambahkan 5 item pakaian favorit Anda sebagai string menggunakan metode push().
3. Hapus pakaian kelima dari array menggunakan metode pop().
4. Tambahkan item pakaian baru menggunakan metode push().
5. Gunakan console.log untuk menampilkan item ketiga dari larik clothes di konsol.
6. Buat literal objek kosong baru dan tetapkan ke variabel favCar.
7. Dengan menggunakan notasi titik, tetapkan properti color ke objek favCar dan berikan nilai string dengan warna pilihan Anda.
8. Dengan menggunakan notasi titik, tetapkan properti convertible ke objek favCar dan berikan nilai boolean sesuai pilihan Anda.
9. Gunakan konsol untuk mencatat seluruh objek favCar.

>**Catatan**: Pada Langkah2 , metode push() menambahkan satu atau beberapa elemen ke akhir larik dan mengembalikan panjang larik yang baru. Sebagai contoh: 

```javascript
let arr = [1, 2];
let newLength = arr.push(3); // Adds 3 to the end
console.log(newLength); // 3 — the new length of the array
console.log(arr); // [1, 2, 3]
```

### Tips
1. Ingatlah untuk menggunakan sintaks literal objek: {}.
2. Ingatlah untuk menggunakan sintaksis literal larik: [].
