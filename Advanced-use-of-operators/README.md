# Exercise: Advanced Use of Operators

## Task 1: Logical && Operator
Buat variabel `score` dengan nilai `8`.  
Gunakan `console.log()` dengan string `"Mid-level skills:"` dan bandingkan `score` di atas `0` dan di bawah `10` menggunakan operator `&&`.

**Output yang diharapkan:**  
`"Mid-level skills: true"`

---

## Task 2: Logical || Operator
Deklarasikan variabel `timeRemaining` dengan nilai `0` dan `energy` dengan nilai `10`.  
Gunakan `console.log()` dengan parameter `"Game over: "` dan `timeRemaining == 0 || energy == 0`.

**Output yang diharapkan:**  
`"Game over: true"`

---

## Task 3: Modulus Operator %
Deklarasikan variabel berikut:
- `num1` = `2`
- `num2` = `5`
- `test1` = `num1 % 2`
- `test2` = `num2 % 2`
- `result1` = `test1 == 0`
- `result2` = `test2 == 0`

Gunakan `console.log()` untuk menampilkan apakah `num1` dan `num2` adalah bilangan genap.

**Output yang diharapkan:**  
`"Is 2 an even number? true"`  
`"Is 5 an even number? false"`

---

## Task 4: Operator +
Tampilkan hasil penjumlahan `5` dan `10` menggunakan operator `+`.

**Output yang diharapkan:**  
`15`

## Task 5: Concatenation dengan Operator +
Deklarasikan variabel:
- `now` = `"Now in "`
- `three` = `3`
- `d` = `"D!"`

Gabungkan ketiga variabel tersebut.

**Output yang diharapkan:**  
`"Now in 3D!"`
