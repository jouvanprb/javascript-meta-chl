# Exercise: Declaring Variables

## Tugas
1. Deklarasikan variabel baru bernama `petDog` dan beri nilai `"Rex"`
2. Deklarasikan variabel baru bernama `petCat` dan beri nilai `"Pepper"`
3. Tampilkan variabel `petDog` ke konsol
4. Tampilkan variabel `petCat` ke konsol
5. Tampilkan teks `"My pet dog's name is: "` dan variabel `petDog`
6. Tampilkan teks `"My pet cat's name is: "` dan variabel `petCat`
7. Deklarasikan variabel `catSound` dan beri nilai `"purr"`
8. Deklarasikan variabel `dogSound` dan beri nilai `"woof"`
9. Tampilkan variabel `petDog`, string `"says"`, dan variabel `dogSound`
10. Tampilkan variabel `petCat`, string `"says"`, dan variabel `catSound`
11. Ubah nilai `catSound` menjadi `"meow"`
12. Tampilkan variabel `petCat`, string `"now says"`, dan variabel `catSound`
